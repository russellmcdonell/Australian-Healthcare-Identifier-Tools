# Australian-Healthcare-Identifier-Tools
Tools to generate valid test Australian Healthcare Identifiers and validate Australian Healthcare Identifers

This repository is a collection of small Javascript algorithms which create or validate Australian Healthcare Identifiers.
The Javascript code is built into mini-web pages and 'index.html' is the landing page for all of them.

There tools have been repackaged and published by Integrating the Healthcare Enterprise (IHE) Australia
(Russell Alexander McDonell's Healthcare Identifier Tools)[https://ramd.iheaustralia.online/]
