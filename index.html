Generate or Validate Austalian Healthcare Identifiers

These tools are for test data only

The Validate tools only check the structure of the identifier and checksum if there is one.
To validate a real Australian Healthcare Identier you should use the Services Australia Health Professional Online Services (HPOS) portal.

The Generate tools generate identifiers for the testing range, where there is one.
Where there is no testing range, the Generate tools makes NO attempt to ensure that none of the generated identifiers match the identifier of any person, living or dead.
Where there is no testing range, the Generate tools makes NO attempt to ensure that none of the generated identifiers are unlikely to mach the identifier of any person, living or dead.
